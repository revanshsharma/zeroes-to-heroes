### Description

This is a Day 2 program to check whether a person is eligible to vote based on their age.

### Files Included

### Output

* If the age entered is less than 0, it prints:
  `Invalid age entered`
* If the age is exactly 18, it prints:
  `You just became eligible to vote`
* If the age is greater than 18, it prints:
  `You are eligible to vote`
* Otherwise, it prints:
  `You are not eligible to vote`
